# PMM Workshop Skills

A set of guided, conversational workshops for product marketers, built on Gartner frameworks. Each workshop asks focused questions one at a time, then synthesizes a polished output document. Skills can pull context from previously completed workshops.

## Recommended Order of Operations

1. **ICP Workshop** (`pmm-workshop-ideal-customer-profile`)
2. **Competitive Intelligence Workshop** (`pmm-workshop-competitive-intelligence`)
3. **Positioning Workshop** (`pmm-workshop-positioning`)
4. **Messaging Workshop** (`pmm-workshop-messaging`)

Each workshop can be run standalone, but running them in this order lets later workshops mine earlier outputs (e.g. Positioning can pull buyer/pain context straight from the ICP, and differentiation context from Competitive Intel) instead of re-asking the same questions. Competitive Intelligence is also designed to run on an ongoing basis — revisit it any time the landscape shifts, not just this one time before Positioning.

---

## 1. ICP Workshop

**What it does:** Guides you through the 6 Gartner ICP attribute blocks — Firmographics, Technographics, Psychographics, Business Situation, Business & Operating Model, and Resources — one question at a time. For each block, it captures the **Ideal**, **Acceptable**, and **Unacceptable** customer profile.

**Output:** A complete Ideal Customer Profile, delivered as a structured profile (table format) plus a narrative summary. Deliverable format is your choice: Word doc, Google Doc, or PDF.

## 2. Competitive Intelligence Workshop

**What it does:** An ongoing workshop for monitoring the competitive landscape — covering competitor teardowns, comparison charts across dimensions, and kill sheets for head-to-head selling situations. Run this before Positioning so differentiation is grounded in real competitive context, and revisit it any time the landscape shifts.

**Output:** Five outputs, always delivered as a combined PDF plus in-chat formats:
- A competitive comparison chart (rendered visual, markdown table, or both)
- A **De-Positioning Guide** — using the Todd Irwin framework, turns each competitor's claimed strength into a liability against 2–3 "Hero Pain Points," rather than just differentiating
- Optional kill sheets — one-pagers per competitor for sales teams

## 3. Positioning Workshop

**What it does:** Guides you through the 6 blocks of the Gartner positioning framework (Buyer, Needs & Wants, Product Name, Category, Compelling Reason to Buy, Differentiation). If an ICP, competitive intel, or other docs are available, it mines them first and only asks about genuine gaps.

**Output:**
- A complete Gartner-format positioning statement
- Block-by-block rationale for stakeholder buy-in
- Your value proposition in three forms (single sentence, problem/solution/outcome, homepage headline + subheadline)
- A guide for activating the positioning across marketing, product, and sales

## 4. Messaging Workshop

**What it does:** Builds a full messaging framework anchored on at least 3 messaging pillars. For each pillar, it works through five fixed rows: key pain solved, key benefit, core messages, common objections, and use cases.

**Output:** A messaging document formatted as a pillar-by-row matrix. Delivered as a Word doc, PDF, or native Google Doc — serves as the single source of truth for messaging across the team.

---

## Notes
- All 4 workshops ask one question at a time and confirm your answers before moving on — no need to answer a wall of questions at once.
- Outputs from earlier workshops (ICP, Positioning, Messaging) can feed into later ones, and into downstream tools like the Sales Enablement Hub builders.
